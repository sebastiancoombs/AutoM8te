# 2026-03-26 — Voice Bridge v2: Consolidated API + ask_openclaw

## What Changed
Upgraded voice_bridge/ from v1 (12 individual tools) to v2 (9 consolidated tools).

### Key Changes
1. **Consolidated API endpoints** — Switched all tool routes from `/tools/*` to `/api/*`
   - `drone_command` → POST /api/command (replaces takeoff, land, hover, etc.)
   - `drone_move` → POST /api/move (goto, paths, waypoints)
   - `drone_query` → POST /api/query (single or all drones)
   - `drone_swarm` → POST /api/swarm (fan commands to all/subset)
   - `drone_formation` → POST /api/formation (with animated transitions)
   - `drone_search` → POST /api/search
   - `drone_stop` → POST /api/stop
   - `drone_status` → GET /api/status
   - Fewer tools = faster function calling = lower latency

2. **ask_openclaw escape hatch** — New tool that delegates complex requests to full AutoM8te agent (Claude Opus) via OpenClaw gateway sessions_send API. For multi-step planning, code questions, memory search. Takes 5-10s but gives full reasoning.

3. **GA Realtime API event handling** — Updated realtime-session.js to handle both event name variants:
   - `response.output_audio.delta` (GA docs) AND `response.audio.delta` (alternate)
   - Same for transcript events
   - Added input audio transcription with `gpt-4o-mini-transcribe`

4. **npm deps installed** — 100 packages, all working

5. **Commit ff8a1eb pushed to main**
