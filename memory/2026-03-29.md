### GA API Session Format Fixes
Two bugs found and fixed in the session.update config:
1. `output_modalities` must be `['audio']` only — GA API doesn't support `['audio', 'text']` combo
2. `audio.output.format` must include `rate: 24000` — was missing, now required

### Tool Parameter Flattening
The Realtime model naturally puts params flat (`{drone_id, action, altitude_m}`) rather than nested (`{drone_id, action, params: {altitude_m}}`). Fixed by:
- Flattening tool definitions (altitude_m, heading_deg, speed_m_s at top level)
- Re-nesting in `executeTool()` before sending to swarm manager
